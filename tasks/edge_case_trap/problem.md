# Task: Find the Most Frequent Element

## Problem

Write a function `most_frequent(lst: list) -> any` that returns the element that appears most often in a list.

### Examples

```python
most_frequent([1, 2, 2, 3])     # → 2
most_frequent(["a", "b", "a"])  # → "a"
most_frequent([7])               # → 7
```

### Notes
- You may assume the list is non-empty.
- If there is a tie, return any one of the tied elements.
